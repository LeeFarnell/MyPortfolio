Projects

P1XEL


This was my second major project as part of the Coding Boot Camp. Working in a team with some of the other students, we created P1XEL, a new video games based application. P1XEL allows the user to search for any video game, regardless of platform, and obtain information about the game, as well as screenshots of the game play. The user can also create their own profile so that they can log in and store games to their digital library, having both a "Have Played" list and a "Want To Play" list. The user can also link their Steam account, so they can see their recent Steam events, as well as their friends recent events, and compare their achievements with their friends.

All of this was made possible with the use of two APIs. The first being the IGDB (Internet Games Database), where we were able to get data from the API and render them into cards, and into their own pages, with various information available to the user. We also used the Steam API which we linked to the users profile, so they could see all of their games, friends and achievements, and also render their profile picture to be used as the profile picture for the site.

Sights and Sounds


This was my first major project as part of the Coding Boot Camp. Working in a team with some of the other students, we created the Sites and Sounds website. Sites and Sounds allows the user to search for any city in the world and search for a venue which will meet their needs, be this a restaurant, a park, a gym etc. We pulled the data from the foursqaure API to render the cards on the page. We also called on the Ticketmaster API which ran as a widget to show events happening within the searched location.

You can check out the completed website by clicking the image above, or head on over to the GitHub repository.

Weather Dashboard


My first venture into APIs was with the Weather Dashboard. As part of the Coding Boot Camp, we were tasked with created a fully functional Weather App, which called on two open weather APIs, to pull weather data for whatever city was searched for by the user. We also had to show the user a 5 day forecast, the UVI index and also icons of what the weather will be like.
