Projects

Sites and Sounds


This was my first major project as part of the Coding Boot Camp. Working in a team with some of the other students, we created the Sites and Sounds website. Sites and Sounds allows the user to search for any city in the world and search for a venue which will meet their needs, be this a restaurant, a park, a gym etc. We pulled the data from the foursqaure API to render the cards on the page. We also called on the Ticketmaster API which ran as a widget to show events happening within the searched location.

You can check out the completed website by clicking the image above, or head on over to the GitHub repository.

Weather Dashboard


My first venture into APIs was with the Weather Dashboard. As part of the Coding Boot Camp, we were tasked with created a fully functional Weather App, which called on two open weather APIs, to pull weather data for whatever city was searched for by the user. We also had to show the user a 5 day forecast, the UVI index and also icons of what the weather will be like.

Want to find out the weather in your area? Click on the image above to view the site, or head on over to the GitHub repository.

Password Generator


As part of the Coding Bootcamp, I was given a preset HTML and CSS page and it was my task to create the javascript element for a password generator. It was my responsibility to ensure that the necessary criteria from the user story was met. This included having alerts and prompts on the page and ensuring that the returned password met the criteria set by the user.
